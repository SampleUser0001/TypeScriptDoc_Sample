# TypeScriptDoc_Sample
TypeDocを使ってみる

## インストール

```sh
npm install --save-dev typedoc
```

## 生成

``` sh
npx typedoc src
```

## 参照

### ローカル

``` sh
# ファイルは自分で作った
docker-compose up
```
